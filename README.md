# Real-Time Chat

A modern, responsive real-time chat application built using **React** and **Firebase**. The app features a sleek user interface designed with **Material UI (MUI)** and handles data synchronization instantly across all connected clients.


## [Live Demo](https://chat-react-a3541.web.app)

### Tech Stack
* **Frontend:** React.js & Material UI
* **Backend:** Firebase (Firestore & Auth)
* **Real-time:** Live data syncing via Firestore
